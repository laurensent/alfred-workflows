# Action Launcher

Browse and launch actions from all your Alfred workflows.

## Features

- Lists all Keywords, Script Filters, External Triggers, and Hotkeys
- Shows action type and parent workflow
- Frecency-based sorting (combines frequency + recency)
- Most used actions appear at the top
- Actions with keyboard shortcuts are prioritized

## Usage

- `wf [query]`: Browse and search actions
- Ctrl+Opt+Shift+Cmd+K: Hotkey to open action list
- Enter: Launch the selected action

## Requirements

- Alfred 5 with Powerpack
