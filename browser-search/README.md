# Browser Search

Search bookmarks and history with fuzzy matching.

## Supported Browsers

- Chrome (including Beta, Canary, Dev)
- Edge
- Brave
- Arc
- Vivaldi
- ChatGPT Atlas

## Usage

```
bm [query]              # Search bookmarks
bm #folder [query]      # Filter by folder
bh [query]              # Search history
bmp                     # List profiles
```

## Shortcuts

| Key | Action |
|-----|--------|
| Enter | Open |
| Cmd+Enter | Incognito |
| Cmd+B | Bookmarks |
| Cmd+H | History |

## Settings

| Option | Description |
|--------|-------------|
| Browser | Select browser |
| Profiles | Profiles to search |
| Fuzzy | Search tolerance (1-5) |
