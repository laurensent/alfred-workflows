# ClipImg

Browse and paste images from Alfred's clipboard history.

## Features

- Preview clipboard history images with thumbnails
- Shows relative time, dimensions, file size, and source app
- Quick paste to active application
- Filters only image files (png, jpg, jpeg, gif, tiff, bmp, webp, heic, svg, ico)
- Smart search: 50 recent items by default, full search when typing

## Usage

- **Keyword**: `clpimg`
- **Hotkey**: Configurable in Alfred Preferences

### Actions

| Key | Action |
|-----|--------|
| Enter | Copy and paste selected image |
| Cmd+Enter | Reveal in Finder |
| Opt+Enter | Open with default app |

## Requirements

- Alfred 5 with Powerpack
- Clipboard History enabled in Alfred
