# Finder Ghostty

Open current Finder path in Ghostty.

## Features

- Quick switch from Finder to Ghostty terminal
- Opens current Finder location in new Ghostty window
- Falls back to Desktop if no Finder window is open

## Usage

`fg`: Open current Finder path in new Ghostty window

## Requirements

- Alfred 5 with Powerpack
- [Ghostty](https://ghostty.org/) terminal
