## GitHub Repos

List and open your GitHub repositories with fuzzy search.

### Usage

**Keyword:** `gh` or Hotkey `Ctrl+Cmd+G`

- Type to fuzzy search your repos
- Enter: Open repo in browser
- Cmd+Enter: Copy repo URL
- Option+Enter: Open commits page

**File Action:** Git Commit & Push
- Auto commit and push for git repos with remote

### Requirements

- GitHub CLI (`gh`) installed and authenticated
